# Copy / Legal Pass — pre-announcement checklist

> Run this against EVERY public-facing string (announcement, homepage, citizen
> page, dashboard, share card, X post) **before** any public activation push.
> This is a self-check to make the lawyer's job cheap — it is **not** legal
> advice. A crypto-aware attorney should still review the final copy + the
> non-refundable-ETH terms + the hex-freeze before paid launch.

---

## 🚫 Must NOT appear anywhere public

- No **profit** language
- No **investment** language
- No **"worth more"**
- No **"appreciating asset"** / "appreciate"
- No **"guaranteed"** (value, demand, returns, anything)
- No **"floor will move"** / "floor protection" / price predictions
- No **"on-chain history"** — the anchor is built but NOT deployed
- No **"fully on-chain agent"** claim (that's Normies' lane, untrue for us)
- No implying **celebrities / honoraries are involved** or endorsing (Vitalik, Elon, etc. — names are tributes, not partnerships)
- No **overclaiming OOGIES** — they're "hidden utility coming", nothing more
- No saying **HEX is money / an investment / redeemable** for anything outside the city
- No **ROI**, **yield**, **passive income**, **alpha**, **"undervalued"**, **"snipe"** framing

## ✅ Approved, safer wording

- **more useful**
- **more developed**
- **trained** (vs **blank**)
- **visible work history**
- **tied to the FREELON**
- **follows the FREELON inside the app**
- **activation stays with the NFT**
- **history stays with the NFT**
- **on-chain anchoring built but not yet deployed**
- **can earn HEX inside the ecosystem**
- **HEX is a shared reward layer, not money** (not redeemable, not an investment)
- **a trained FREELON is different from a blank one**

## Quick scan (run before posting)

A fast grep against the banned terms catches most slips:
```
grep -inE "worth more|appreciat|investment|profit|guaranteed|floor will|on-chain (history|agent)|undervalued|snipe|passive income|ROI|yield|alpha" <files>
```
(Allow "value proposition" / "perceived value" only when the AGENT is analyzing a
buyer's view in its output — never as a claim the project makes.)

## 🔒 PROOF-POST GATE — do NOT post publicly until ALL true

```
1. Redeploy is complete (the paid-depth + copy-clean fixes are live)
2. Real-wallet Red Team test passes
3. Output is sharp (would a skeptical holder pay 0.005 ETH for it?)
4. Runs decrement correctly (40 → 39)
5. Résumé updates
6. Work-log updates
7. Share card updates
8. This copy/legal checklist is passed
```

Only when every box is checked does the proof post go out — and the post itself
must pass the wording rules above.

The honest frame for the post: **They proved people want NFTs to wake up.
FREELONS prove activated agents actually work.** Show the real Red Team output;
don't claim momentum you don't have.
