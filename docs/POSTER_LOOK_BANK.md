# FREELON CITY — Poster Look-Bank (Cinematic Variety Library)

Date: 2026-06-09. The breadth library for the creation engine. **Variety is the product** — no two
looks should feel the same. Each entry is an allowlist template:
`{ id, label, headline?, lookPrompt }`. The engine composes: `{citizen identity} + {lookPrompt} +
{reserved text area} + "shareable X image, no gore/sexual/hate, no real-person likeness"`.

Citizen identity glue (keep recognisable, vary everything else): *a hooded faceted citizen with a
single glowing hex eye*. Swap world, era, lighting, medium freely. Captions: status/lore only, never
financial.

## How to use / grow this
- Each `id` becomes a server-allowlist entry (moderation-safe; no free client text).
- Compose at render time: citizen identity + `lookPrompt` + headline + reserved text area.
- **Grow aggressively** — this bank is meant to expand. Want 20 more in any category (more neon
  variants, more Crypt tombs, seasonal/event looks)? Ask and I'll extend. Breadth = more posts.
- Tune which looks fit which collection by feel; nothing here is fixed — swap freely.

## Guardrails (the only limits)
No gore / sexual / hateful imagery · no real-person likeness or copyrighted characters · captions
carry no financial/return claims · keep Crypt TCG a side lane. Everything aesthetic is open.
